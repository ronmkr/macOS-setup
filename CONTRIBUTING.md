# Contributing to macOS Setup Framework

We love your help in making this the best macOS automation tool!

## How to Contribute
1. **Port a Module**: Pick a setting from the old Bash scripts and port it to a new Go struct and module.
2. **Improve Configuration**: Add more descriptive comments or better defaults to `config/settings.toml`.
3. **Bug Fixes**: Help us handle edge cases in different macOS versions.

## Development Process
1. Fork the repo.
2. Create a feature branch.
3. Use `make build` and `make dry` to test your changes.
4. Ensure your code follows Go standards (`go fmt`).
5. Submit a Pull Request!
