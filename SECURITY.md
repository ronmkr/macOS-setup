# Security Policy

## Reporting a Vulnerability
We take the security of this project seriously. If you discover a vulnerability, please do not disclose it publicly. 

Please report security issues by:
1. Opening a GitHub Issue with the "Security" label.
2. We will investigate and respond as soon as possible.

## Supported Versions
Only the latest version of the `macos-setup` framework is supported for security updates.

## Guidelines
- This tool modifies system settings; always run a `make dry` before applying.
- We never log or store your sudo password.
